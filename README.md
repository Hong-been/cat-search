
# VanillaJS로 구현한 컴포넌트 기반의 고양이검색 SPA
- 프로그래머스 과제관 "고양이 검색 사이트"를 풀고, 해당 API를 응용하여 만든 프로젝트입니다.
- 바닐라JS를 사용하여 클래스 컴포넌트 방식을 구현하여 사용했습니다.
- 성능최적화를 위해 이미지에 lazy loading을 적용하였습니다.

<br>

- 상단의 다크모드 토글버튼과 랜덤고양이 버튼이 있습니다.
